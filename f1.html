School Management System

School Management Systemthe main objective of the School Management System is to manage the details of Schools, Students, Classes, Teachers, Registrations. It manages all the information about Schools, Cources, Registrations, Schools. The project is totally built at administrative end and thus only the administrator is guaranteed the access. The purpose of the project is to build an application program to reduce the manual work for managing the Schools, Students, Cources, Classes. It tracks all the details about the Classes, Teachers, Registrations.School management software or also referred to as School ERP is an interface through which school staff can manage their specific tasks which revolve around a student’s life-cycle. Be it authorities to maintain this data in digital form.By using all these features admin can add and remove staff and student details, Student fee, Library details, online test, account report, class routine, attendance, Exam schedule, new notifications on notice board and send messages. Also, they provide the average daily attendance, monthly attendance, monthly SMS report.
